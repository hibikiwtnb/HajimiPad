# 哈基米私人音效板（Hajimi Soundboard）

## 项目说明
本项目是一个网页端「搞笑音效板」，外观与页面骨架需参考工程目录中的 `MP4Tools.html`，但核心功能替换为点击 PAD 方块播放音效。

目标风格为：
- 工具站外观（简洁、留白、多信息密度低）
- 内容为娱乐 / 整活用途
- 页面结构清晰，便于后期扩展

---

#### 主功能区：PAD 音效板（核心）

##### PAD 总体结构
- 使用多组 **3×3 PAD 面板**
- 当前至少 3 组，一组内可以不满9个按钮
- 各组内3个一行
- 每组前显示分组标题

##### 单个 PAD 按钮结构
每个 PAD 为一个正方形按钮，结构如下：
[ 图片（长宽比1:1的正方形，可选） ] 
如果没有图片则是
[ 音效名称（在按钮中央居中显示） ]

- 图片：当前还没准备好
- 文字：在按钮中央居中显示，默认为音频文件的文件名（不包括后缀名）

## 交互反馈要求

### Hover（悬停）
- 仅边框变色（数 px）
- 不使用缩放、阴影或动画

### Press（点击）
- 整体覆盖一层半透明颜色
- 无需按压感、无需位移

### 播放状态
- 不需要播放中高亮
- 不显示进度
- 假设音效大多为短音效


## 音频播放规则
- 默认允许多个音效叠加播放
- 不限制快速连续点击


## 页脚（Footer）
- 极简
- 功能性说明为主
- 风格与 MP4Tools 工具站一致


## 明确不做的事项
- 不展示 MIDI / 键盘 / 乐器 等未来功能
- 不显示“施工中 / 敬请期待”
- 不引入复杂动画或重型前端框架
- 不改变 MP4Tools 风格的页面骨架
